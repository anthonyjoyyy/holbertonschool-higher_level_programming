# Python - Imports & Modules

This project is part of the Holberton School curriculum and focuses on learning how to use imports and modules in Python.

## Description

The goal of this project is to understand:

- How to import functions from another file
- How to use modules in Python
- How to create your own modules
- How to use command line arguments
- Basic Python program structure

## Technologies Used

- Python 3
- Ubuntu Linux
- Visual Studio Code

## Files

Each file in this repository contains exercises and tasks related to Python imports and modules.

## Learning Objectives

By completing this project, I learned:

- How Python modules work
- Different ways to import functions and variables
- How to execute Python files
- How to organize code across multiple files

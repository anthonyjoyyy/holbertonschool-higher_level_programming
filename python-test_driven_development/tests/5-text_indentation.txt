===============================
How to use the text_indentation module
===============================

First, import the function from the module:
>>> text_indentation = __import__('5-text_indentation').text_indentation

Test case 1: Normal usage with punctuation marks
>>> text_indentation("Hello. World? This is: a test.")
Hello.
<BLANKLINE>
World?
<BLANKLINE>
This is:
<BLANKLINE>
a test.

Test case 2: Verifying that multiple spaces after punctuation are skipped
>>> text_indentation("Sentence one.    Sentence two?")
Sentence one.
<BLANKLINE>
Sentence two?

Test case 3: Passing a non-string argument should raise a TypeError
>>> text_indentation(12345)
Traceback (most recent call last):
TypeError: text must be a string

Test case 4: Missing the text argument entirely should raise Python's native TypeError
>>> text_indentation()
Traceback (most recent call last):
TypeError: text_indentation() missing 1 required positional argument: 'text'
